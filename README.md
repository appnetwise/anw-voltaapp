
# anw-voltacab

anw-voltacab is hybrid mobile application


## Installation

Install dependancy with npm

```bash
  npm i
```
Install ionic globally
``` bash
npm i -g @ionic/cli
```

Start the application
```bash
ionic serve
```
## Generating APK file
### On windows machine
Insatll cordova globally
```
npm i -g cordova
npm i -g cordova-res
```
```bash
ionic platform add android
```
